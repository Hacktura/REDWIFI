# Code of Conduct

## Ethical and Legal Use

Red WiFi is a professional penetration testing tool. All users agree to use this framework:

### Only for Authorized Testing

- **Obtain written permission** before testing any network you do not own
- Test only networks where you have explicit authorization
- Do not test networks you do not own or have permission to test
- Respect all applicable laws and regulations

### Legal Compliance

Users are responsible for complying with:
- Local, state, and federal laws
- Computer fraud and abuse regulations
- Telecommunications regulations
- Employer policies
- Network owner policies

### Responsible Disclosure

If you discover vulnerabilities:
1. Do not publicly disclose before notifying the affected party
2. Give reasonable time for patching (30-90 days)
3. Report through responsible disclosure channels
4. Work constructively with vendor/owner

## Professional Conduct

### Be Respectful

- Treat all community members with respect
- Welcome diverse perspectives
- Provide constructive feedback
- Resolve conflicts professionally

### Be Professional

- Use appropriate language
- Avoid harassment or discrimination
- Respect confidentiality
- Follow industry standards

### Be Responsible

- Test carefully and thoroughly
- Document your work
- Minimize impact on tested networks
- Report findings professionally

## Security Research

Red WiFi is designed for authorized security research:

- **Academic Research**: Use in educational environments with proper oversight
- **Professional Testing**: Use only with signed penetration testing agreements
- **Lab Environment**: Use in isolated test networks
- **Authorized Testing**: Never test without explicit written permission

## Unacceptable Behavior

Prohibited activities include:
- Unauthorized network testing
- Stealing passwords or credentials
- Accessing systems without permission
- Disrupting network service
- Selling exploits or vulnerabilities
- Malicious hacking
- Harassment or discrimination
- Violating privacy or confidentiality

## Reporting Violations

Report code of conduct violations to: security@redwifi.dev

## Community Standards

We are committed to:
- Safe and respectful community
- Ethical use of security tools
- Professional standards
- Transparency and honesty
- Legal compliance

## Disclaimer

Red WiFi is provided for authorized security testing only. Users are responsible for their actions. The authors assume no liability for misuse or damage caused by this framework.

By using Red WiFi, you agree to:
- Only test networks you own or have written permission to test
- Follow all applicable laws and regulations
- Comply with this code of conduct
- Accept full responsibility for your actions
